# ProjectAS
# 모험 시뮬레이션 (Adventure Simulation)
장르: 시뮬레이션
래퍼런스: Lushfoil Photography Sim

Elevator Pitch
  홀로 여러 곳을 탐험하면서 그곳에서 보물을 찾는 힐링 및 시뮬레이션 게임

코어 판타지
  극도로 사실적이고 아름다운 자연의 ASMR을 통해 혼자만의 휴식과 힐링을 제공.
  인간의 발길이 닿지 않은 숨겨진 절경이나 잊혀진 유적을 발견했을 때의 경이로움 제공.

핵심 메커니즘
환경 관찰 및 탐지 도구 : 맵의 절경을 보면서 보물을 찾기 위한 손전등, 금속 탐지기, 나침반 등의 탐험 장비를 활용.
비폭력적 환경 퍼즐 : 적의 전투가 아예 없으며, 환경을 이용한 퍼즐로 자연스럽게 길이 열리거나 자연적인 미로를 탈출하는 듯한 퍼즐 시스템.
감각적인 이동 : 이동 과정 자체가 피로한 노동이 아닌 맵을 돌아다니는 행위 자체에 재미를 부여.
보물 전시 공간 : 발견한 보물을 개인 전시관에 전시할 수 있도록 수집욕 자극.
